# Prends ton dessin en main
## Blog / Forum / Client de messagerie / Application de création, gestion et partage de dessin
Si vous souhaitez vous lancer dans le dessin mais que vous n’osez pas, je vous montre dans ce site mon apprentissage autodidacte point par point avec mes dessins personnels, et je vous donne les outils pour vous lancer ainsi qu’une plateforme d’échange et d’entraide.
## Fonctionnalités
### Fonctionnalités utilisateur
* S'inscrire
* Se connecter
* Demander un nouveau mot de passe automatique par mail
* Écrire à l'administrateur par mail
* Envoyer et recevoir messages d'un autre utilisateur via messagerie
### Blog
* *Slider* les dessins avec leur description
* Accéder à tous les dessins et leur description (listés avec barre de navigation)
### Forum
* Poster des sujets, écrire des messages
* Se connecter/s'enregistrer pour le faire et être redirigé vers la vue initiale du forum
### Dessin
* Dessiner via l'IHM
* Télécharger le dessin
* L'envoyer directement dans l'espace perso
* Uploader un dessin de la machine locale dans l'espace perso
### Dessins de l'utilisateur
* *Slider* les dessins avec leur description
* Accéder à tous les dessins et leur description (listés avec barre de navigation)
* Télécharger
* Supprimer
* Renommer
* Commenter
* Rendre public ou privé le dessin
* Supprimer le commentaire
* Accéder aux dessins publics des autres utilisateurs
